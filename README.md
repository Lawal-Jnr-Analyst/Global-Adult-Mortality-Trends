# 🌍 Global Adult Mortality Trends (2016–2021)

## 📊 Overview
This project analyzes global adult mortality trends using data from the World Health Organization (WHO). The dashboard provides insights into how mortality rates have changed over time, highlighting regional disparities and identifying countries with the highest mortality burden.

It is designed to support data-driven understanding of global health outcomes and reveal where progress has been made and where challenges persist.

## 🎯 Objectives
- Analyze global adult mortality trends from 2016 to 2021
- Compare mortality rates across different regions
- Identify regions with the highest and lowest mortality burden
- Highlight countries with the highest adult mortality rates
- Provide an interactive dashboard for exploration and insights

## 🛠 Tools Used
Power BI – Data visualization and dashboard creation
Power Query – Data transformation and cleaning
DAX – Measures, KPIs, and calculations

## Key Insights
- Global adult mortality declined steadily between 2016 and 2021
- The overall reduction of 6.59% indicates gradual improvement in global health outcomes
- Africa consistently records the highest mortality rates across all years
- Europe maintains the lowest mortality rates, reflecting stronger health systems
- The gap between regions remains wide, showing persistent global health inequality
- Countries such as Lesotho, Eswatini, Central African Republic, Zimbabwe, and Mozambique have the highest mortality burden
- Regional trends show slight improvements, but not enough to close the disparity gap

## 🧹 Data Cleaning & Preparation
- Removed incomplete and inconsistent records
- Standardized country and region names
- Validated numerical data types
- Created calculated measures for % change and regional comparisons
